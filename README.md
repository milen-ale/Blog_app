
## Blog_APP 
This is a blog application built with Ruby On Rails. On completion the app will enable users to post, to comment and like on posts.

## Built With
- Ruby
- Ruby on Rails
- Rubocop

## Getting Started
To get a local copy run the following steps:
Prerequisites
- Ruby
- Ruby on rails
- Postgresql
- Install stylelint
- rubocop
- Run tests
- run rspec test
